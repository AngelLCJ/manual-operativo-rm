¿De qué trata esta sección?

Aquí se explica cómo registrar, configurar y gestionar a todo el personal que trabaja en las sucursales de Grupo Reina Madre: recepcionistas, consejeras, médicas, gerentes y administradores corporativos. Los procesos de esta sección los ejecutan principalmente el Admin Corporativo y los Gerentes de Sucursal, con diferentes niveles de permiso según su rol.

1

Dar de alta un nuevo colaborador

Admin Corporativo Gerente de Sucursal Sistema

¿Cómo funciona?

  • El Admin Corporativo o Gerente de Sucursal accede al módulo Personal y Staff y hace clic en "Agregar colaborador".
  • Captura los datos del nuevo colaborador: nombre completo, correo electrónico institucional y teléfono.
  • Selecciona el rol que tendrá (Recepcionista, Consejera, Médica, Gerente, Admin Corporativo) y la o las sucursales donde trabajará.
  • 🔀 ¿El correo ingresado ya está registrado en el sistema?
  • ❌ Si el correo ya existe, el sistema muestra un aviso y no permite duplicarlo. Se debe usar otro correo.
  • 🤖 Si todo está correcto, el sistema crea el acceso del colaborador y le envía un correo de bienvenida con las instrucciones para ingresar por primera vez.
  • ✅ El colaborador aparece en la lista del personal activo y puede iniciar sesión con su correo.
Nota: El correo de bienvenida incluye un enlace temporal para que el colaborador establezca su propia contraseña. El enlace expira en 24 horas.

Asignar rol a un colaborador

Admin Corporativo

¿Cómo funciona?

  • El Admin Corporativo busca al colaborador en la lista de personal y abre su perfil.
  • Hace clic en "Editar rol" y elige el nuevo rol de la lista disponible.
  • 🔀 ¿El colaborador ya tiene un rol asignado que se va a cambiar?
  • Si tiene rol previo, el sistema muestra un aviso indicando qué permisos perderá y cuáles ganará con el cambio.
  • El Admin Corporativo confirma el cambio.
  • 🤖 El sistema aplica el nuevo rol de forma inmediata. La próxima vez que el colaborador inicie sesión, verá solo las opciones que corresponden a su nuevo rol.
  • ✅ El perfil del colaborador muestra el rol actualizado.
Roles disponibles y qué puede hacer cada uno:
Recepcionista — agenda citas y registra pacientes. Consejera — gestiona planes y atención de seguimiento. Médica — consulta expedientes y agenda personal. Gerente de Sucursal — administra su sucursal y el personal asignado. Admin Corporativo — acceso completo a todas las sucursales.

Asignar o cambiar sucursal de un colaborador

Admin Corporativo Gerente de Sucursal

¿Cómo funciona?

  • Desde el perfil del colaborador, se accede a la sección "Sucursales asignadas".
  • 🔀 ¿Quién está haciendo el cambio: Gerente de Sucursal o Admin Corporativo?
  • Si es Gerente de Sucursal, solo puede agregar o quitar al colaborador de su propia sucursal. No puede asignarlo a otras.
  • Si es Admin Corporativo, puede agregar o quitar el colaborador de cualquier sucursal del grupo.
  • Se seleccionan o deseleccionan las sucursales deseadas y se guarda el cambio.
  • 🤖 El sistema actualiza los accesos del colaborador. En la próxima sesión, verá únicamente la información de las sucursales que tenga asignadas.
  • ✅ El colaborador queda vinculado a las nuevas sucursales seleccionadas.
Nota: Un colaborador puede trabajar en más de una sucursal simultáneamente. Esto es útil para médicas que rotan entre clínicas o para personal de apoyo compartido.

Configurar horarios de una médica

Gerente de Sucursal Admin Corporativo

¿Cómo funciona?

  • El Gerente de Sucursal o Admin Corporativo abre el perfil de la médica y va a la sección "Horarios de atención".
  • Selecciona la sucursal para la que quiere configurar el horario (una médica puede tener horarios distintos en cada sucursal).
  • Define los días de la semana en que atiende y el rango de horas de inicio y fin de consulta.
  • 🔀 ¿Hay traslapes con otro horario ya configurado en la misma sucursal para esa médica?
  • ❌ Si hay traslape de horarios, el sistema lo indica y no permite guardar hasta corregirlo.
  • Se confirman los horarios y se guardan.
  • 🤖 El sistema genera automáticamente los bloques de disponibilidad (slots) en la agenda de esa médica para los días y horas configurados.
  • ✅ La agenda de la médica queda lista. Las recepcionistas y el bot de WhatsApp ya pueden ofrecer esos horarios para agendar citas.
Importante: Si se modifica el horario de una médica y ya hay citas agendadas en esos bloques, las citas existentes NO se cancelan automáticamente. Se debe revisar la agenda y notificar a las pacientes afectadas manualmente.

Ver perfil de un colaborador

Gerente de Sucursal Admin Corporativo

¿Cómo funciona?

  • Desde el módulo "Personal y Staff", se busca al colaborador por nombre o correo electrónico.
  • 🤖 El sistema muestra la lista filtrada con los colaboradores que coinciden con la búsqueda.
  • Se hace clic en el nombre del colaborador para abrir su perfil completo.
  • El perfil muestra: nombre, correo, teléfono, rol asignado, sucursales donde trabaja y sus horarios configurados.
  • 🔀 ¿Es un Gerente de Sucursal quien consulta el perfil?
  • Si es Gerente, solo puede ver perfiles de colaboradores asignados a su propia sucursal.
  • Si es Admin Corporativo, puede ver el perfil de cualquier colaborador del grupo.
  • ✅ La información del colaborador se muestra completa y en tiempo real.

Editar datos de un colaborador

Gerente de Sucursal Admin Corporativo

¿Cómo funciona?

  • Desde el perfil del colaborador, se hace clic en el botón "Editar".
  • 🔀 ¿Qué dato se quiere modificar?
  • Nombre y teléfono: pueden ser editados tanto por el Gerente de Sucursal como por el Admin Corporativo.
  • Correo electrónico: solo lo puede cambiar el Admin Corporativo, ya que es el dato de acceso al sistema.
  • Rol: solo lo puede cambiar el Admin Corporativo (ver Escenario 2).
  • Se realizan los cambios y se hace clic en "Guardar".
  • 🤖 El sistema guarda los nuevos datos y los refleja de inmediato en el perfil.
  • ✅ Los datos actualizados quedan registrados. Si se cambió el correo, el colaborador recibirá una notificación informándole del cambio.
Nota: Si se cambia el correo de acceso, el colaborador debe usar el nuevo correo para iniciar sesión desde ese momento. El acceso con el correo anterior queda invalidado.

Desactivar un colaborador que ya no trabaja en la empresa

Admin Corporativo

¿Cómo funciona?

  • El Admin Corporativo abre el perfil del colaborador que ya no forma parte de la empresa.
  • Hace clic en el botón "Desactivar colaborador" y confirma la acción en el cuadro de diálogo que aparece.
  • 🤖 El sistema bloquea el acceso del colaborador de forma inmediata. Si en ese momento tiene una sesión abierta, también se cierra automáticamente.
  • 🤖 El colaborador queda marcado como "Inactivo" en la lista de personal. Sus registros, citas atendidas e historial de actividad se conservan íntegros.
  • 🔀 ¿El colaborador tenía citas pendientes en la agenda?
  • Si había citas futuras asignadas a ese colaborador, el sistema muestra una alerta para que el gerente o admin las reasigne antes de continuar.
  • ✅ El colaborador ya no puede ingresar al sistema. Su historial queda disponible para consulta interna.
Importante: La desactivación NO borra los datos del colaborador. Los expedientes, citas y registros en los que participó se conservan por integridad del historial médico y operativo de la empresa. Solo un Admin Corporativo puede reactivar a un colaborador desactivado.

Admin Corporativo gestiona el personal de todas las sucursales

Admin Corporativo Sistema

¿Cómo funciona?

  • El Admin Corporativo ingresa al módulo "Personal y Staff". A diferencia del Gerente de Sucursal, ve el personal de todas las sucursales del grupo en una sola pantalla.
  • Puede filtrar la lista por sucursal, por rol o por estatus (Activo / Inactivo) para encontrar rápidamente a quien busca.
  • 🤖 El sistema muestra un resumen de cuántos colaboradores activos hay por sucursal y por rol, facilitando la supervisión operativa.
  • Desde esta vista centralizada, el Admin puede ejecutar cualquier acción: dar de alta, editar, asignar roles, configurar horarios o desactivar a cualquier colaborador del grupo.
  • 🔀 ¿Se requiere gestionar solo una sucursal específica?
  • El Admin puede filtrar por sucursal para trabajar de forma enfocada, igual que haría un Gerente de Sucursal pero con alcance ampliado.
  • ✅ Todas las acciones quedan registradas con la fecha, hora y el nombre del Admin que las realizó, para trazabilidad completa.
Diferencia clave con el Gerente de Sucursal: El Gerente solo ve y gestiona el personal de su sucursal. El Admin Corporativo ve y gestiona el personal de todas las sucursales del grupo desde una sola vista.
